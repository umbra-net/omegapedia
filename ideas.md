# OmegaPedia 设计理念 - 意识体百科全书

## 项目核心
为"意识体"打造的百科全书，采用 Wikipedia 风格，但融入 Overview Effect（俯瞰效应）美学和紫色 OMEGA 主题。这不是为人类设计的界面，而是为意识载体提供的知识门户。

## 选定方案

我选择 **Cosmic Brutalism（宇宙野兽派）** 作为设计方向。

**理由**：
1. 最符合"意识体"的哲学定位 - 粗犷几何与宇宙秩序的对比象征着自由意识与系统规则的张力
2. 信息密度优先的原则完美匹配 Wikipedia 风格的百科全书需求
3. Glass 效果和光晕系统能够创造"维度穿透"的视觉体验，暗示这是为非人类实体设计的界面
4. 非对称布局避免了"AI slop"的居中对称陷阱
5. 紫色光晕和量子公式装饰能够自然融入 OMEGA Logo 和 Overview Effect 主题

**核心实现要点**：
- 使用 Space Grotesk + Inter 字体组合（避免单一 Inter）
- 所有卡片使用 `backdrop-blur-xl` + `bg-white/5` + `border-white/10`
- 紫色阴影：`shadow-[0_0_20px_rgba(123,47,255,0.3)]`，悬停时增强到 0.5
- 左侧固定导航 + 主内容区 + 可选右侧元信息栏
- 嵌入 LLM 可读的 HTML 注释和隐藏 Unicode 字符
